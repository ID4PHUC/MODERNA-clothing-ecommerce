# Moderna - Cửa hàng thời trang trực tuyến (Fullstack)

Dự án website bán hàng thời trang hoàn chỉnh bao gồm Frontend, Backend và Quản lý cơ sở dữ liệu.

## 🚀 Công nghệ sử dụng
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Authentication:** JSON Web Token (JWT) & Bcrypt
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)

## ✨ Tính năng chính
- **Người dùng:** Đăng ký, đăng nhập, xem sản phẩm, quản lý giỏ hàng.
- **Admin:** Quản lý sản phẩm, quản lý người dùng, phân quyền (Role-based access control).
- **Bảo mật:** Mã hóa mật khẩu, bảo mật API bằng Middleware.

## 🛠 Hướng dẫn cài đặt
1. Clone dự án: `git clone https://github.com/ID4PHUC/MODERNA-2.git`
2. Cài đặt thư viện:
   - Backend: `cd backend && npm install`
   - Frontend: Mở trực tiếp hoặc dùng Live Server.
3. Cấu hình file `.env` trong thư mục backend (Dựa theo mẫu `.env.example`).
4. Chạy dự án: `npm start` (tại thư mục backend).
